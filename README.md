# 架空のこども園のサイトです
HTML、CSS、JavaScriptを使用して、架空のこども園のサイトを作成しました。

- 配色は優しい雰囲気を出すため暖色使用しています。
- 見出しはリボンの装飾をつけて可愛らしさが出るようにしました。
- "生活"の項目はタブの機能を使い、「１日の流れ」と「年間行事」を切り替えることができます。
- ”FAQ”の項目はやアコーディオン機能をつけて見やすいように工夫しています。
- ”アクセス”にはグーグルマップを読み込みます。
